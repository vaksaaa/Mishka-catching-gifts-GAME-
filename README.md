# Mishka-catching-gifts-GAME-
What do you get when you steal a fat cat’s presents and give a cannon to his rival? PURE MAYHEM! Navigate through a storm of bombs and survival kits as you try to endure the most bizarre feline feud ever. It’s fast, it’s loud, and it’s raining medkits!

🐱 Fat Cat vs. Cannon Cat 🐱
🛠 Technical Info

    ⚙️ Engine: Made with ClickTeam Fusion 2.5.

    🖋 Font Required: Before starting the game, you must install the DS Crystal.ttf font found in the game folder.

🎮 Controls

    ⌨️ Movement: Use A and D keys to move.

    🖱 Alternative: You can also use the Mouse Buttons to control your character!

🍎 Gameplay Mechanics

    💊 Health Packs: If you can't pick up a medkit, it means you're already fully healed or the item isn't glowing green yet.

    💣 Chaos: Watch out! A barrage of bombs and medkits will be constantly flying your way.

⚖️ The Honesty Test

    📂 Save File: Inside the game folder, you’ll find an unprotected save file.

    🏆 Your Choice: You can manually edit it to give yourself as many points as you want—the game is testing your integrity!
